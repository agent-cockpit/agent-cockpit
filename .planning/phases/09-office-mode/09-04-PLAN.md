**Step 2 — Update OfficePage.tsx to make tests GREEN:**

In the `sessions.map(...)` loop (lines 59-73), extend the computation block to derive both values before the `return`:

```typescript
{sessions.map((session, i) => {
  const sessionEvents = events[session.sessionId] ?? []
  const lastEvent = sessionEvents.at(-1)
  const agentState = deriveAgentState(session, lastEvent)
  const elapsedMs = Date.now() - Date.parse(session.startedAt)
  const lastToolUsed =
    lastEvent?.type === 'tool_call' ? (lastEvent.toolName as string | undefined) : undefined
  return (
    <AgentSprite
      key={session.sessionId}
      session={session}
      agentState={agentState}
      position={getPosition(session.sessionId, i)}
      isDragging={activeDragId === session.sessionId}
      onClick={() => handleSpriteClick(session.sessionId)}
      elapsedMs={elapsedMs}
      lastToolUsed={lastToolUsed}
    />
  )
})}
```

Note: `lastEvent.toolName` is cast to `string | undefined` because NormalizedEvent is a discriminated union — TypeScript knows toolName exists on tool_call events but the type narrowing may not propagate through `.at(-1)` without a type assertion.
