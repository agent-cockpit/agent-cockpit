## Decisions Made

- `vi.mock('node:child_process')` at module level is required for ESM — `vi.spyOn` cannot redefine namespace properties on ESM modules (TypeError: Cannot redefine property)
- Used `void (async () => {...})()` IIFE pattern inside `req.on('end')` to enable `await` without changing the outer function signature
- ClaudeLauncher.launch() is a stub in this plan — full spawn implementation is Plan 02
- Shared workspacePath existence check lives in `handleLaunchSession` before the provider branch; ClaudeLauncher.preflight() duplicates it so the class is self-contained when called standalone
