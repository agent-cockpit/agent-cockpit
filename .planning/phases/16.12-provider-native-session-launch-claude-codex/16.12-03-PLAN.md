    1. New SubmitState type:
       ```typescript

         | { type: 'waiting_for_session_start'; sessionId: string }
         | { type: 'error'; message: string }
       ```
       Delete claude-success and codex-success variants entirely.

    2. handleSubmit: after successful POST response (mode: 'initiated'), set state to `{ type: 'waiting_for_session_start', sessionId: data.sessionId }`. Remove all hookCommand handling.

    3. Session detection useEffect:
       ```typescript
       useEffect(() => {
         if (state.type !== 'waiting_for_session_start') return;
         const { sessionId } = state;

         // Watch Zustand store for session to appear
         const unsubscribe = useStore.subscribe(
           (s) => s.sessions[sessionId],
           (session) => {
             if (session) { handleClose(); }
           }
         );

         // 30-second timeout
         const timer = setTimeout(() => {
           setState({ type: 'error', message: 'Session launch timed out. Check daemon logs and try again.' });
         }, 30_000);

         return () => { unsubscribe(); clearTimeout(timer); };
       }, [state.type === 'waiting_for_session_start' ? state.sessionId : null]);
       ```
       Use `useStore.subscribe` with selector (Zustand v4 pattern — second arg is selector callback).

    4. Render waiting_for_session_start state:
       ```tsx
       {state.type === 'waiting_for_session_start' && (
         <div className="space-y-4">
           <p className="[font-family:var(--font-mono-data)] text-xs text-muted-foreground">
             Waiting for session to start…
           </p>
           <p className="cockpit-label text-[0.65rem]">Session ID: {state.sessionId}</p>
           <div className="flex justify-end">
             <button type="button" onClick={handleClose} className="cockpit-btn">Cancel</button>
           </div>
         </div>
       )}
       ```

    5. Remove all claude-success JSX block (copy-command pre, Copy button). Remove codex-success JSX block.

    DO NOT add manual terminal instructions or copy-command UX.
    DO NOT change the form inputs (provider select + workspace path only).
