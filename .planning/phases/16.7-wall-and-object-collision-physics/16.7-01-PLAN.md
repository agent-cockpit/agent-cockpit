```json

  "cells": [{ "x": -46, "y": -43, "terrainId": 1 }, ...]

// terrainId 1 = hull floor (walkable)
// terrainId 4 = patrol line (walkable)
// terrainId 5 = scuffed panels (walkable)
```

```json
    }
  ]
}
// Objects with description starting "Character:" → non-solid (pass-through)
// Objects with visible: false → skip
// All others with visible: true → SOLID furniture
// boundingBox coords are already in pixel-space (0,0 = map top-left) — no transform needed
```

    Key implementation details:
    - Constants: TILE_SIZE=32, TILE_ORIGIN_X=-46, TILE_ORIGIN_Y=-43, SOLID_TERRAIN_IDS=new Set([2,3])
    - solidTiles: Set&lt;string&gt; — keys are `"${tileX},${tileY}"` in tile-space (not pixel-space)
    - solidObjects: Rect[] — pixel-space bounding boxes from manifest
    - loadTerrain(data: { defaultTerrain: number; cells: Array&lt;{x: number; y: number; terrainId: number}&gt; }):
      - Clear solidTiles

    - loadObjects(objects: Array&lt;{boundingBox:{x:number;y:number;width:number;height:number}; description:string; visible:boolean}&gt;):
      - Clear solidObjects
      - Skip if !obj.visible or obj.description.startsWith('Character:')
      - Push { x, y, w:width, h:height } from obj.boundingBox
    - overlaps(x, y, w, h): boolean:
      - Check 4 corners: (x,y), (x+w-1,y), (x,y+h-1), (x+w-1,y+h-1)
      - For each corner: tx=Math.floor(cx/TILE_SIZE), ty=Math.floor(cy/TILE_SIZE); if solidTiles.has(`${tx},${ty}`) return true
      - For each obj in solidObjects: standard AABB overlap test; if overlapping return true
      - Return false
