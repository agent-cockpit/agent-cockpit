**Non-terminal PTY Claude tabs now recover from replay and empty-store hydration using structured session history rather than assuming a pre-populated live session slice.**

- Added PTY mode to UI session summaries so panel rendering can preserve transport context after hydration.
- Updated `TimelinePanel` to display PTY mode details and `DiffPanel` to infer active/ended status from replayed session events when the session store is empty.
- Updated `MemoryPanel` to recover workspace context from replayed `session_start` events so `CLAUDE.md`, notes, and suggestions still load after popup reopen.
- Updated `ArtifactsPanel` messaging and regression tests so PTY Claude activity remains visible instead of silently flattening to generic state.
- Verified the focused PTY UI suite is green across approvals, timeline, diff, memory, and artifacts.
