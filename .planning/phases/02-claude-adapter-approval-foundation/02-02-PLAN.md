       - Export class ApprovalQueue:
         - register(approvalId: string, event: NormalizedEvent, db: Database.Database): void
           1. Extract { actionType, riskLevel, proposedAction, affectedPaths, whyRisky } from event (ApprovalRequestEvent fields)
           2. Call insertApproval(db, ...) SYNCHRONOUSLY
           3. Call eventBus.emit('event', event) — broadcast happens AFTER insert
           4. Add approvalId to pendingSet
         - decide(approvalId: string, decision: 'approve' | 'deny' | 'always_allow', db: Database.Database): void
           1. If approvalId not in pendingSet → return (no-op)
           2. pendingSet.delete(approvalId) — claim atomically
           3. const status = decision === 'approve' ? 'approved' : decision === 'deny' ? 'denied' : 'always_allow'
           4. updateApprovalDecision(db, approvalId, status, 'User decision')
           5. If always_allow: insertAlwaysAllowRule(db, { sessionId from event, toolName, pattern, createdAt })
           6. Emit ApprovalResolvedEvent on eventBus: { type: 'approval_resolved', decision: status, approvalId, ... }
           7. Call resolveApproval(approvalId, decision === 'approve' ? 'allow' : decision === 'deny' ? 'deny' : 'allow')
         - handleTimeout(approvalId: string, db: Database.Database): void
           1. If approvalId not in pendingSet → return (no-op)
           2. pendingSet.delete(approvalId)
           3. updateApprovalDecision(db, approvalId, 'timeout', 'Auto-denied: timeout exceeded')
           4. Emit ApprovalResolvedEvent on eventBus: { type: 'approval_resolved', decision: 'timeout', approvalId, ... }
           5. Call resolveApproval(approvalId, 'deny', 'Auto-denied: timeout exceeded')
       - Export a singleton: export const approvalQueue = new ApprovalQueue()
