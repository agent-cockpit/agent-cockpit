  <action>
    Ask the user about each decision area, presenting specific options from the research.
    Wait for their answer before moving to the next question.

    **Q1: Space sub-theme**
    Show options (from references doc):
    A) Space Station Operations — ISS/mission control feel, blues/grays/orange accents, functional/military
    B) Deep Space / Sci-Fi — dark purples and blues, neon highlights, mysterious and dramatic
    C) Retro Space — limited CGA/EGA palette, dithering, nostalgic arcade feel
    D) Cyberpunk Space — neon greens/purples/pinks on black, hacker aesthetic
    Which resonates most with the "Agent Mission Control" brand?

    **Q2: Agent character type**
    The agent sprite is the most important asset (it represents Claude Code/Codex sessions).
    Options:
    A) Astronaut — humanoid in space suit, classic and relatable
    B) Robot/Android — mechanical, precise, fits the "AI agent" concept
    C) Hologram/Ghost — glowing translucent humanoid, futuristic and ethereal
    D) Alien crew member — playful, unique, unexpected
    Should there be 1 character type (all agents same) or 2 (Claude different from Codex)?

    **Q3: Character size and directions**
    A) 32px, 4 directions (front/back/left/right) — standard, fits current CSS
    B) 32px, single direction (front-facing only) — simpler, cheaper
    C) 64px, 4 directions — more detail, bigger on screen
    Note: current AgentSprite renders at 32x32 with imageRendering: pixelated.

    **Q4: State differentiation style**
    How should the 8 states be visually distinct? From references:
    A) Animation-first — each state has a different animation (idle bounce vs walk cycle vs flash)
    B) Color-first — same idle animation, character color/glow changes per state
    C) Mixed — key states have unique animations (blocked = shake, completed = victory), rest use color
    Remember: blocked and completed are the most important states to differentiate.

    **Q5: Office Mode background**
    Should the canvas behind the agent sprites have a background?
    A) Starfield — simple parallax stars, classic space
    B) Space station grid floor — tiled floor tiles, feels like a real workspace
    C) Dark plain — no background, focus on agents (current behavior)
    D) Nebula — atmospheric color gradient with star particles

    **Q6: Budget tier priorities**
    From 10.2-03, Tier 3 (nice-to-have) uses the remaining budget.
    Which Tier 3 assets matter most to you?
    A) Office background tiles
    B) UI accent/border elements
    C) Icon set for panel headers
    D) Loading/connecting animations
