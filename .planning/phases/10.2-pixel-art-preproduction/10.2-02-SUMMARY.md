## Decisions Made

- **Option A Deep Space Terminal palette recommended** over alternatives B/C/D — best balance of dark-mode aesthetics, state differentiation, and professional legibility for a developer tool context
- **Three-layer state signal** (color + icon + pose) rather than relying on animation alone — ensures state is readable even when animation is not in peripheral view
- **blocked state must be visually dominant** — more alarming than any other state because it requires immediate human response
- **completed state uses no animation** — stillness is a strong visual signal that separates "done" from all active states
- **PixelLab anchor-prompt approach** — generate one canonical base character first; all state variants reference it as a delta to maintain visual consistency across generations
