    Call get_character(character_id) with include_preview:true every 10 seconds until status != 'processing'.
    If status='failed', retry create_character once with ai_freedom:400. Log character_id to scripts/generation-log.md.

    STEP 3 — Generate idle animation (shared for all 5 color states):
    Call animate_character with:
      character_id: [from step 2]
      template_animation_id: "breathing-idle"
      directions: ["south","north","east","west","south-east","south-west","north-east","north-west"]
    Poll get_character until idle animation frames appear.

    STEP 4 — Generate blocked animation:
    Call animate_character with:
      character_id: [from step 2]
      template_animation_id: "fight-stance-idle-8-frames"
      directions: ["south","north","east","west","south-east","south-west","north-east","north-west"]
    Fallback if template fails: use "scary-walk". Poll until complete.

    STEP 5 — Generate completed animation:
    Call animate_character with:
      character_id: [from step 2]
      template_animation_id: "jumping-1"
      directions: ["south","north","east","west","south-east","south-west","north-east","north-west"]
    Fallback: "two-footed-jump". Poll until complete.

    STEP 6 — Generate failed animation:
    Call animate_character with:
      character_id: [from step 2]
      template_animation_id: "falling-back-death"
      directions: ["south","north","east","west","south-east","south-west","north-east","north-west"]
    Fallback: "taking-punch". Poll until complete.

    STEP 7 — Download all frames:
    For each animation state [idle, blocked, completed, failed] and each of 8 directions, download every frame URL returned by get_character to:
      assets/raw/astronaut/{state}/{direction}/frame-0.png, frame-1.png, ...
    Use fetch() to download. Each frame must be a non-zero PNG file.
    IMPORTANT: Measure actual frame pixel dimensions on first download. Log actual size to generation-log.md.
    If dimensions are not 64×64 (expected ~90×90 due to canvas inflation), the buildSheet script handles cropping via sharp resize with fit:'contain'.

    STEP 8 — Assemble sprite sheet:
    Run: npx tsx scripts/build-spritesheet.ts astronaut
    This reads assets/raw/astronaut/**/*.png, builds 32-row × maxFrames-col sheet, writes:
      packages/ui/public/sprites/astronaut-sheet.png
      packages/ui/public/sprites/astronaut-manifest.json
